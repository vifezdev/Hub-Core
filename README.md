# vifez hub core
Free, Lightweight hub core

- Queues (Custom & Portal)
- Rank core support (Aqua, Vault, Glacial)
- Scoreboard
- Hotbar
- Server selector
- Configurable
- Build mode
- Spawn system
- Reload configs
- Player join / leave message 
- Flat file data saving
- Double jump
